# iPet

#### twa-camp-2016集体项目

##### 运行完整过程
1. git clone git@github.com:bmeritter/iPet.git
2. sudo mongod
3. npm run refreshMongo
4. npm run wpw
5. npm start
6. http://localhost:3000
